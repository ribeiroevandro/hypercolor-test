# Hypercolor Docs

Hypercolor is a gallery and playground for Tailwind CSS gradients. The documentation in this folder now follows the product routes directly, so each document maps to a page users can actually open in the app.

## Documented routes

- `/` — this index and the homepage experience
- `/generator` — standard gradient generator
- `/mesh` — mesh gradient generator
- `/grainy` — grainy gradient generator
- `/flags` — world flag gradient gallery

## Homepage experience

The homepage is the discovery-first part of Hypercolor. It presents a grid of curated gradients from the core dataset and gives each card its own controls.

## What users can do on the homepage

- browse the curated gradient collection
- change the direction of each gradient card
- copy Tailwind classes from a card
- copy CSS from a card
- download a gradient as an image

## Related route docs

- `generator.md` explains the fully custom standard gradient workflow
- `mesh.md` explains the multi-stop mesh editor
- `grainy.md` explains the textured gradient workflow
- `flags.md` explains the static flag-inspired gallery
